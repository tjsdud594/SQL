[초보자도 할 수 있는 SQL - 류선영]

1. 입사일(hire_date)이 06년인 employees들 중 연봉(salary)이 5000 이상인 직원들의 이름(first,last)과 연봉정보를 추출하여 오름차순 정렬하세요.

모범 답안 select hire_date,first_name,last_name,salary from employees where substr(hire_date,0,2)='06' and salary>'5000' order by salary asc; HIRE_DAT FIRST_NAME LAST_NAME SALARY -------- ---------------------------------------- -------------------------------------------------- ---------- 06/11/03 Sarath Sewall 7000 06/12/09 Nanette Cambrault 7500 06/03/07 Jose Manuel Urman 7800 06/03/30 Christopher Olsen 8000 06/04/23 Jack Livingston 8400 06/03/24 Jonathon Taylor 8600 06/01/03 Alexander Hunold 9000 06/01/24 Tayler Fox 9600 06/03/23 Harrison Bloom 10000

2. 작업 내역(job_history)에서 소요기간이 1400일 이상인 직원들의 employee_id 값을 추출하세요.(소요기간 column을 생성하시오.)

모범 답안 select employee_id,round(end_date-start_date) as 소요기간 from job_history where round(end_date-start_date) > '1400'; EMPLOYEE_ID 소요기간 -------------------- ---------- 102 2018 101 1497 201 1401 200 2100 200 1644

3. 부서번호가 60인 직원의 풀네임과 급여가 높은 순서대로 내림차순 정렬하시오.

모범 답안 select first_name, last_name, salary from employees where department_id = 60 order bysalary desc; FIRST_NAME LAST_NAME SALARY ---------------------------------------- -------------------------------------------------- ---------- Alexander Hunold 9000 Bruce Ernst 6000 David Austin 4800 Valli Pataballa 4800 Diana Lorentz 4200

4. 근무년수가 18년이 넘어가는 직원의 풀네임을 근무년수가 높은 순으로 출력하라.

모범 답안 select first_name, last_name, round(round(sysdate-hire_date)/365, 1) as 근무년수 from employees where round(round(sysdate-hire_date)/365, 1) > 18 order by 근무년수 desc; FIRST_NAME LAST_NAME 근무년수 ---------------------------------------- -------------------------------------------------- ---------- Lex De Haan 20.4 Shelley Higgins 19 William Gietz 19 Susan Mavris 19 Hermann Baer 19 Nancy Greenberg 18.8 Daniel Faviet 18.8 Den Raphaely 18.5 Payam Kaufling 18.1 Alexander Khoo 18.1

5. countries table의 대륙 정보(region_id)를 1은 'Europe', 2는 'America', 3은 'Asia', 4는 'Africa_continent'로 바꾼 후 이들을 글자수가 적은 순으로 정렬해서 출력하세요.
